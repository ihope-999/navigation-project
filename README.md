# Navigation Project

> **This project is ongoing and not yet complete.**

A Hybrid A* path planning system for car-like vehicles, written in C++. The planner finds a drivable path through a 2D map while respecting the car's steering constraints.

## Tech

- C++
- PGM occupancy grid maps

## Features

- Hybrid A* planner
- Car collision detection using a rectangular vehicle model
- PGM map loading
